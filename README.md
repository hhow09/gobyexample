## Develop Notes
### Add Examples
1. create feature branch
2. add example to `examples.txt` `<title>|<folder-name>`
3. run `./tools/build` before push to remote
4. create Pull Request
5. Site deployment will trigger after merged to `master` branch 

### Local preview
1. `./tools/build`
2. `./tools/serve`
